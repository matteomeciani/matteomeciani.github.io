## Features

- **Modern Design**: Clean, professional interface with gradient backgrounds and smooth animations
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Interactive Elements**:
  - Smooth scrolling navigation
  - Animated timeline for education history
  - Hover effects on cards and buttons
  - Scroll-triggered animations
  - 3D tilt effects on project cards
- **Performance Optimized**: Pure HTML, CSS, and JavaScript (no heavy frameworks)
- **Accessibility**: Semantic HTML and keyboard navigation support

## Technology Stack

- **HTML5**: Semantic markup
- **CSS3**: Custom properties, Flexbox, Grid, animations
- **JavaScript**: Vanilla JS for interactivity and animations
- **Fonts**: Inter (Google Fonts)
- **Deployment**: GitHub Pages

## Customization

To customize this website for your own use:

1. Update personal information in `index.html`
2. Replace `profile.jpg` with your own photo
3. Modify colors in `styles.css` by changing CSS custom properties in `:root`
4. Add or remove sections as needed
5. Update project descriptions and links

## Local Development

To run locally:

1. Clone this repository
```bash
git clone https://github.com/matteomeciani/matteomeciani.github.io.git
```

2. Open `index.html` in your browser
```bash
open index.html
```

No build process or dependencies required!

## Deployment

This site is automatically deployed via GitHub Pages from the `main` branch.

To update the live site:
```bash
git add .
git commit -m "Your update message"
git push origin main
```

Changes will be live within a few minutes.
